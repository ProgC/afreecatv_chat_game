afreecatv_chat_game
===================

Afreeca TV which is a broadcasting service in Korea. the SDK and server program let the user play the game via chatting.

아프리카 TV는 한국에서 인터넷 방송을 하기 위한 서비스 입니다. 아프리카에서 제공하는 SDK를 이용해 채팅 내용을
캡쳐하고 그 내용을 서버로 보낸 후 이 서버가 현재 화면에 떠 있는 게임에 입력을 보내 원격으로 게임을 즐길 수 있게
하는 시스템 입니다.

사용 방법
1. afreeca_SDK에 존재하는 Greeting예제를 빌드하여 Greeting.lpi를 얻어 냅니다. 그리고 이 파일을
아프리카 TV가 설치 되어 있는 폴더에 들어가 Plugins폴더에 복사합니다.

2. 아프리카 스튜디오를 띄우기 전에 AfreecaReceiver에 존재하는 서버를 실행 시킵니다.

3. 서버를 실행 시킨 후 아프리카 스튜디오를 띄우면 그때 부터 서버와 통신을 시작합니다.

4. 만일 현재 에뮬레이터가 켜져 있다면 입력한 내용이 에뮬레이터로 전송 됩니다.
